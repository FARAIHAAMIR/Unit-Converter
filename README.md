UNIT CONVERTER
---
Project Report

1. Project Description

The goal of this project is to develop a console-based Unit Converter application in C language. The program enables users to perform precise and efficient unit conversions across various categories, including length, temperature, time, weight, speed, area, angle, and number systems.

This project demonstrates core programming concepts such as modular design, user-defined functions, conditional statements, loops, arrays, and efficient user interaction.
---
2. Project Functionalities

1. Conversion of Length Units:
Convert between common units like feet, inches, miles, centimeters, meters, kilometers, and more.


2. Conversion of Temperature Units:
Supports conversions between Kelvin, Celsius, and Fahrenheit scales.


3. Conversion of Time Units:
Converts between seconds, minutes, and hours.


4. Conversion of Weight Units:
Handles conversions for kilograms, grams, and pounds.


5. Conversion of Speed Units:
Converts between kilometers per hour, miles per hour, and meters per second.


6. Conversion of Area Units:
Supports units like square meters, square kilometers, and acres.


7. Conversion of Angle Units:
Includes degrees, radians, and gradians.


8. Number System Conversion:
Converts numbers between binary, decimal, hexadecimal, and octal systems.

---

3. Data Types Description

float input, output: Variables to store the user input value and calculated output.

int choice: Stores the user's selected category or subcategory of conversion.

char continue_program: Character variable to determine if the user wants to perform another conversion.
